# 科研假设生成器 - 可视化演示

## 🎯 功能说明

这是一个端到端的可视化演示，展示科研假设生成器的完整工作流程。

## 📁 文件说明

### 数据文件
- `static/data/hypothesis_generator.json` - 假设生成器配置（变量池、策略、模板）
- `static/data/hypothesis_example.json` - 完整的运行示例数据
- `static/data/causal_graph.json` - 原始因果图谱
- `static/data/causal_graph_v2.json` - 改进版因果图谱

### 可视化界面
- `hypothesis_viewer.html` - 假设生成过程演示（推荐）
- `causal_graph_viewer.html` - 因果图谱可视化

## 🚀 启动方式

### 方式1：使用 Python 服务器（推荐）

```bash
cd sandbox
python start_viewer.py
```

或者双击 `start_viewer.bat`

### 方式2：直接打开 HTML

直接在浏览器中打开 `hypothesis_viewer.html`

## 📊 演示内容

### 6个步骤的完整流程

1. **用户输入** - 研究领域和目标
2. **意图分析** - 提取关键词和研究意图
3. **变量匹配** - 匹配相关的自变量、因变量、调节变量
4. **文献检查** - 识别已验证和未探索的因果路径
5. **假设生成** - 基于6种策略生成5个假设
6. **排序推荐** - 评分排序并生成分析方案

### 示例场景

**用户输入**：量子计算技术

**生成的假设**：
- H1: 理论迁移（新颖性 0.7）
- H2: 未探索路径（新颖性 0.85）⭐ 推荐
- H3: 边界条件（新颖性 0.65）
- H4: 中介机制（新颖性 0.75）
- H5: 交互效应（新颖性 0.8）

**推荐假设**：H2 - 国际合作对技术影响力的影响

**分析方案**：包含5个DAG任务，从数据准备到结果可视化

## 💡 创新点展示

### 6种假设生成策略

1. **理论迁移** - 在新领域验证已有理论
2. **路径探索** - 发现文献空白
3. **边界条件** - 揭示理论适用范围
4. **中介机制** - 打开因果"黑箱"
5. **反事实推理** - 发现反直觉结果
6. **交互效应** - 探索协同作用

### 新颖性评分

- 0.9: 反直觉发现（最高）
- 0.8-0.85: 未探索路径、交互效应
- 0.7-0.75: 理论迁移、中介机制
- 0.6-0.65: 边界条件
- 0.2-0.3: 已充分研究（基准）
